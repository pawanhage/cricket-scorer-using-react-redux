## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/pawanhage/cricket-scorer-using-react-redux.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
